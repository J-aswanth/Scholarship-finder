## Technologies Used

---



##  Getting Started

### Backend Setup
```bash
cd backend
npm install
npm start
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### MongoDB Setup
- Install MongoDB locally or use [MongoDB Atlas](https://www.mongodb.com/atlas).
- Create a `.env` file in the `backend` directory and add your MongoDB connection string:
```env
DB_URI=your_mongodb_connection_string
PORT=5000
```
